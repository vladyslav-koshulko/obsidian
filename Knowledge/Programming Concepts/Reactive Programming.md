#concepts #reactive

Реактивное программирование — это парадигма, использующая асинхронное программирование для обработки данных в реальном времени. Она чаще всего применяется в корпоративных Java-приложениях с микроархитектурами. Программирование отслеживает события для запуска изменений «состояния», которые распространяются по всему приложению, и делает это без блокировки ввода-вывода. Это обеспечивает высокую производительность и параллелизм, что крайне важно для крупных приложений, обслуживающих миллионы пользователей.
