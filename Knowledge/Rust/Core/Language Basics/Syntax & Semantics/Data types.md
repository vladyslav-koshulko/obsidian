# Type table

|                           | Типи                                       | Літерали                       |
| ------------------------- | ------------------------------------------ | ------------------------------ |
| Цілі числа зі знаком      | `i8`, `i16`, `i32`, `i64`, `i128`, `isize` | `-10`, `0`, `1_000`, `123_i64` |
| Беззнакові цілі числа     | `u8`, `u16`, `u32`, `u64`, `u128`, `usize` | `0`, `123`, `10_u16`           |
| Числа з плаваючою комою   | `f32`, `f64`                               | `3.14`, `-10.0e20`, `2_f32`    |
| Скалярні значення Unicode | `char`                                     | `'a'`, `'α'`, `'∞'`            |
| Логічні значення          | `bool`                                     | `true`, `false`                |



Типи мають наступну ширину:
- `iN`, `uN` і `fN` мають ширину _N_ біт,
- `isize` і `usize` – це ширина вказівника,
- `char` має ширину 32 біти,
- `bool` має ширину 8 біт.

Усі підкреслення у числах можна опускати, вони призначені лише для розбірливості. Отже, `1_000` можна записати як `1000` (або `10_00`), а `123_i64` можна записати як `123i64`.

**Rust за замовчуванням використовує тип `i32`**

# Типы соединений
_Составные типы_ позволяют объединять несколько значений в один тип. В Rust есть два примитивных составных типа: кортежи и массивы.

## [Тип кортежа](https://doc.rust-lang.org/book/ch03-02-data-types.html#the-tuple-type)
Кортеж _—_ это общий способ группировки нескольких значений различных типов в один составной тип. 
- Кортежи имеют фиксированную длину: после объявления они не могут увеличиваться или уменьшаться в размере.
- Кортеж создается путем записи списка значений, разделенных запятыми, внутри скобок.
- Каждая позиция в кортеже имеет свой тип, и типы различных значений в кортеже не обязательно должны совпадать.
```
fn main() {
    let tup: (i32, f64, u8) = (500, 6.4, 1);
}
```

Мы также можем получить доступ к элементу кортежа напрямую, используя точку ( `.`) с последующим индексом значения, к которому мы хотим получить доступ. Например:
```
fn main() {
    let x: (i32, f64, u8) = (500, 6.4, 1);

    let five_hundred = x.0;

    let six_point_four = x.1;

    let one = x.2;
}
```
Эта программа создает кортеж `x`, а затем обращается к каждому элементу кортежа, используя соответствующие индексы. Как и в большинстве языков программирования, первый индекс в кортеже равен 0.
Кортеж без значений имеет специальное имя — _unit_ . Это значение и соответствующий ему тип записываются `()`и представляют собой пустое значение или пустой возвращаемый тип. Выражения неявно возвращают значение unit, если они не возвращают никакого другого значения.

##  [Тип массива](https://doc.rust-lang.org/book/ch03-02-data-types.html#the-array-type)

Ещё один способ создать коллекцию из нескольких значений — использовать _массив_ .

- каждый элемент массива должен иметь один и тот же тип.
- массивы в Rust имеют фиксированную длину
- Массивы полезны, когда вам нужно разместить данные в стеке,  а не в куче
- когда вы хотите гарантировать, что у вас всегда будет фиксированное количество элементов
- Если вы не уверены, использовать ли массив или вектор, скорее всего, вам следует использовать вектор.
- Значения в массиве записываются в виде списка, разделенного запятыми, внутри квадратных скобок
```
fn main() {
    let a = [1, 2, 3, 4, 5];
}
```

Вектор — это аналогичный тип коллекции, предоставляемый стандартной библиотекой, размер которого _может_ увеличиваться или уменьшаться, поскольку его содержимое находится в куче.

Тип массива указывается в квадратных скобках: тип каждого элемента, точка с запятой, а затем количество элементов в массиве, вот так:
```
let a: [i32; 5] = [1, 2, 3, 4, 5];
```

Также можно инициализировать массив, указав для каждого элемента одинаковое значение, задав начальное значение, затем точку с запятой, а затем длину массива в квадратных скобках, как показано здесь:
```
let a = [3; 5];
```

### [Доступ к элементам массива](https://doc.rust-lang.org/book/ch03-02-data-types.html#array-element-access)

Массив — это единый блок памяти известного фиксированного размера, который может быть выделен в стеке. Доступ к элементам массива осуществляется с помощью индексации, например, так:
```
fn main() {
    let a = [1, 2, 3, 4, 5];

    let first = a[0];
    let second = a[1];
}
```
