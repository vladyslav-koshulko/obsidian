#rust #core #syntax #constants

_константы_ — это значения, которые привязаны к имени и не могут изменяться
- нельзя использовать `mut` `constants` (константы всегда неизменяемы)
- обявляються с помощю `const` 
- тип данных должен быть указан явно
- могут быть обявлены в любой области видимости
- задаються в виде постоянного выражения а не результата выполнения в рантайме
- принято именовывать константы заглавными буквами и нижними подчеркиваниями между словами

```
const THREE_HOURS_IN_SECONDS: u32 = 60 * 60 * 3;
```
