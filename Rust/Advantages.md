Безпека памяті під час компіляції
- без неініціалізованих змінних
- без подвійних звільнень значень
- без використання після звільнення
- без вказівників NULL
- без забутих заблокованих мютексів
- без race condition між потоками
- без недійсності ітератора

Можливості сучасної мови
- переліки та зістввлення шаблонів
- узагальнені типи
- [[Foreign Function interface (FFI)]] без накладних витрат
- Абстракції без витрат
- Чудово деталізовані помилки компілятора з підказками
- Вбудований менеджер залежностей
- Вбудована підтримка тестування
- Чудова підтримка протоколу мовного сервера ([[Language Server Protocol (LSP)]])
