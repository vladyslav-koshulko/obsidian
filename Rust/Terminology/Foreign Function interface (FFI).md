[Foreign Function Interface (FFI)](https://www.google.com/search?q=Foreign+Function+Interface+%28FFI%29&sourceid=chrome&ie=UTF-8&mstk=AUtExfB_1eiPda9QsLDum4zGolnBgYk_Ak52XfWIyyg3cStn-yEtrw5YdaqwiwLSzt0Q5iEQztJYiNcuxf5rVnloUZcS_VLyhQA0zrjHWkgE96X3lM9esYrNrE5u7W6qsFfWmi4M_3MWmCqFiWzzUPXdmhh_gjjPd7zZayTm4ltSSbDLKFo&csui=3&ved=2ahUKEwiJ9-WewraTAxVZiP0HHROODXAQgK4QegQIARAB) — це ==механізм, який дозволяє програмі, написаній однією мовою (наприклад, JavaScript у Deno/Bun або Python), викликати функції та використовувати бібліотеки, написані іншою мовою, зазвичай C або Rust==. FFI забезпечує взаємодію між мовами для використання системних API, підвищення продуктивності або доступу до native-бібліотек.
